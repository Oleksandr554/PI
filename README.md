# **MySpending - Aplikacja Mobilna**
Wersja mobilna MySpending to potężne narzędzie do zarządzania finansami osobistymi, zaprojektowane z myślą o szybkości i wygodzie użytkowania w ruchu. Przenosi ona pełną funkcjonalność systemu do kieszeni, zachowując spójność wizualną i logikę znaną z wersji desktopowej.
